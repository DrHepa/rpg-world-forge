# M2 implementation audit — 2026-07-19

## Outcome

The M2 narrative slice is implemented across source validation, compiler,
typed worldpack loader, deterministic reducer, renderer/UI, persistence,
replay, analyzer, schemas, neutral fixture, and headless tests.

## Closed risks

| ID | Severity | Risk | Resolution |
|---|---|---|---|
| M2-01 | High | Narrative JSON could become an arbitrary execution surface. | Conditions/effects use strict allowlists; no eval, scripts, templates, or provider calls. |
| M2-02 | High | A speaker could reveal facts it cannot know. | Runtime node gate, forbidden learning guard, source validation, and static leak analysis. |
| M2-03 | High | Quest/event cycles could run forever. | Stable ordering and a bounded fixed-point transition loop. |
| M2-04 | High | Narrative state could disappear across saves/replays. | Save/replay format 2 serializes and validates the complete M2 state/action set. |
| M2-05 | Medium | Dialogue graphs could strand the player. | Manual-exit contract plus unreachable/hard/conditional softlock analysis. |
| M2-06 | Medium | Scenes could retrigger every tick. | Persisted once/per-day occurrence keys and paused overlay handling. |
| M2-07 | Medium | Social values could grow without bound. | Loader range checks and runtime clamping to `-100..100`. |
| M2-08 | Medium | Oversized packs could exhaust memory. | 64 MiB worldpack input limit before JSON parsing. |

## Remaining limits

- The static analyzer is conservative rather than a full state-space model.
- Dialogue and quest graphs do not yet support nested boolean expressions or
  multi-target branching stages.
- Quest logs expose active objectives, not a complete player-facing journal.
- Accessibility and localization QA for long dialogue layouts need real-window
  testing per generated game.
- Inter-map travel, economies, needs, goals, construction, and delayed systemic
  consequences remain M3 scope.

## Release evidence

The release gate requires formatting/lint, all headless tests, release source
validation, a zero-warning neutral narrative analysis, a reproducible checked-in
worldpack, save/replay coverage, and a runtime AI-import audit with zero
findings.
